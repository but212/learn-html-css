CSS 선택자

전체 선택자

CSS에서 전체 선택자(*)는 모든 형태의 모든 요소를 선택하여 스타일을 적용할 수 있습니다

요소 선택자

CSS에서 요소 선택자는 HTML 요소 이름 즉 태그명을 사용하여 선택하고 스타일을 적용할 수 있습니다.

클래스 선택자

CSS에서 클래스 선택자는 요소의 클래스 명을 선택하여 스타일을 적용할 수 있습니다.

아이디 선택자

CSS에서 아이디 선택자는 요소의 아이디 명을 선택하여 스타일을 적용할 수 있습니다.

속성 선택자

속성 선택자는 요소의 속성의 유뮤나 속성 값의 조건에 따라 스타일을 적용할 수 있습니다.
